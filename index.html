Float

É uma Tag utilizada para posicionar texto ao redor de uma imagem.

Já foi utilizado para posicionar outros elementos, mas isso caiu em desuso. Hoje em dia só é utilizado para posicionar texto.

O float faz com que um elemento fique flutuando na página.
